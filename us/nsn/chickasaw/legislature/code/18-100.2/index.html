Chickasaw Nation Code

Section 18-100.2 FINDING.

The Chickasaw Tribal Legislature finds that a need exists to provide a system for the Chickasaw Tribal Legislature to utilize in making decisions pertaining to real property owned by, held or acquired for the Chickasaw Nation.

(TL11-002, 12/17/93.)
